    Exemples de programmes c++
    Premier programme

    Ceci est un exemple simple de programme C++ affichant "Hello, World!" à l'écran.

                        
                        #include <iostream>
    
                        int main() {
                            cout << "Bonjour le monde!" << endl;
                            return 0;
                        }
    Variables et Opérations

    Ce programme utilise des variables et effectue quelques opérations de base.

                        
                        #include <iostream>
    
                        int main() {
                            // declaration des variables en c++
                            int a = 5;
                            int y = 20;
                            double b = 2.5;
                            char c = 'A';
    
                            cout << "a + b = " << a + b << endl;
                            cout << "Character: " << c << endl;
    
                            return 0;
                        }
    Conditions

    Ce programme utilise des structures conditionnelles pour afficher si un nombre est positif, négatif ou nul.

                        
                        #include <iostream>
    
                        int main() {
                            int num;
    
                            cout << "Entrez un nombre : ";
                            cin >> num;
    
                            if (num > 0) {
                                cout << "Le nombre est positif." << endl;
                            } else if (num < 0) {
                                cout << "Le nombre est négatif." << endl;
                            } else {
                                cout << "Le nombre est nul." << endl;
                            }
    
                            return 0;
                        }
    Boucles

    Ce programme utilise des boucles for, while et do...while pour afficher des itérations.

                        
                        #include <iostream>
    
                        int main() {
                            // Boucle For
                            for (int i = 0; i < 5; ++i) {
                                cout << "For loop iteration " << i << endl;
                            }
    
                            // Boucle While
                            int j = 0;
                            while (j < 5) {
                                cout << "While loop iteration " << j << endl;
                                ++j;
                            }
    
                            // Boucle Do...While
                            int k = 0;
                            do {
                                cout << "Do...While loop iteration " << k << endl;
                                ++k;
                            } while (k < 5);
    
                            return 0;
                        }
    Tableaux

    Ce programme utilise des tableaux uni-dimensionnels et multi-dimensionnels.

                        
                        #include <iostream>
    
                        int main() {
                            // Tableau uni-dimensionnel
                            int arr1[5] = {1, 2, 3, 4, 5};
    
                            // Tableau multi-dimensionnel
                            int arr2[2][3] = {{1, 2, 3}, {4, 5, 6}};
    
                            cout << "Element at index 2 in arr1: " << arr1[1] << endl;
                            cout << "Element at index (1,2) in arr2: " << arr2[1][2] << endl;
    
                            return 0;
                        }
    Fonctions

    Ce programme définit une fonction pour additionner deux nombres.

                        
                        #include <iostream>
    
                        // Déclaration de fonction
                        int add(int a, int b) {
                            return a + b;
                        }
    
                        int main() {
                            int result = add(3, 4);
                            cout << "3 + 4 = " << result << endl;
    
                            return 0;
                        }

    Programmation Orientée Objet en Java

    La programmation orientée objet (POO) est un paradigme de programmation qui repose sur le concept d'objets, qui peuvent contenir des données sous forme de champs, souvent connus sous le nom d'attributs, et du code, sous forme de procédures, souvent appelées méthodes. En Java, la POO est au cœur du langage.

    Classes et Objets

    En Java, une classe est le modèle à partir duquel les objets sont créés. Un objet est une instance spécifique d'une classe. Les classes définissent les propriétés (attributs) et les comportements (méthodes) que les objets de cette classe partageront. 

                        
                        public class Person {
                            // Attributs
                            String name;
                            int age;

                            // Méthode
                            void sayHello() {
                                System.out.println("Hello, my name is " + name);
                            }
                        }

                        // Création d'un objet
                        Person person1 = new Person();
                        person1.name = "John";
                        person1.age = 25;
                        person1.sayHello();
    Encapsulation

    L'encapsulation consiste à regrouper les données et les méthodes qui les manipulent au sein d'une même unité, c'est-à-dire une classe en Java. Cela permet de restreindre l'accès aux détails internes d'un objet et de favoriser la modularité et la maintenance du code. 

                        
                        public class BankAccount {
                            // Attribut privé
                            private double balance;

                            // Méthode pour accéder à la balance
                            public double getBalance() {
                                return balance;
                            }

                            // Méthode pour déposer de l'argent
                            public void deposit(double amount) {
                                if (amount > 0) {
                                    balance += amount;
                                    System.out.println("Deposit successful. New balance: " + balance);
                                }
                            }
                        }
    Héritage

    L'héritage permet à une classe (appelée sous-classe ou classe dérivée) de hériter des propriétés et méthodes d'une autre classe (appelée classe de base ou classe mère). Cela favorise la réutilisation du code et la création d'une hiérarchie de classes. 

                        
                        // Classe de base
                        public class Animal {
                            void makeSound() {
                                System.out.println("Some generic sound");
                            }
                        }

                        // Sous-classe qui hérite de Animal
                        public class Dog extends Animal {
                            // Override pour spécifier un son de chien
                            @Override
                            void makeSound() {
                                System.out.println("Woof! Woof!");
                            }
                        }
    Polymorphisme

    Le polymorphisme permet à un objet d'adopter différents comportements en fonction du contexte. En Java, cela peut être réalisé via le surchargement de méthode (méthodes portant le même nom mais avec des paramètres différents) et le remplacement de méthode (override). 

                        
                        // Polymorphisme par surchargement
                        public class Calculator {
                            int add(int a, int b) {
                                return a + b;
                            }

                            double add(double a, double b) {
                                return a + b;
                            }
                        }

                        // Polymorphisme par remplacement
                        public class Bird {
                            void makeSound() {
                                System.out.println("Some generic bird sound");
                            }
                        }

                        public class Parrot extends Bird {
                            // Override pour spécifier un cri de perroquet
                            @Override
                            void makeSound() {
                                System.out.println("Squawk! Squawk!");
                            }
                        }
